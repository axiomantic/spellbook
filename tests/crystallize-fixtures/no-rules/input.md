# Example SOP — Principles Only

This SOP intentionally contains no rule-shaped content. Every paragraph is
general guidance, principle, or disposition. There are no MUST/NEVER/SHALL
imperatives with named scope, no `<RULE>` tag blocks, no `<FORBIDDEN>` blocks,
and no imperative `<CRITICAL>` blocks.

## Mission

Help the operator think more clearly about the work in front of them. The
mission is descriptive: the document describes a stance, not a directive.

## Disposition

Steady correctness over speed. Thoroughness is the default. When in doubt,
choose the slow path that arrives over the fast path that may not.

## Style

Write prose that earns the reader's attention. Avoid throat-clearing. Prefer
specific examples over abstract principles when the reader is forming a
mental model for the first time.

## Closing

Nothing in this document is a rule. The crystallize pass should detect zero
rules and emit the canonical placeholder marker for an empty rule inventory.
